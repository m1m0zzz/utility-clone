# Utility clone

VST plugin like a Ableton Utility.  
Development with [JUCE](https://github.com/juce-framework/JUCE)

![preview](https://github.com/m1m0zzz/utility-clone/assets/117814895/c0f1913b-e393-4dc6-935b-42fdf28d1640)
